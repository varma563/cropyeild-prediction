# cropyeild-prediction
Crop Yield Prediction using IBM AutoAI

1️⃣ Project Overview
Crop Yield Prediction is a Machine Learning project developed using IBM Cloud Pak for Data. The system predicts crop yield based on agricultural parameters such as rainfall, temperature, and soil conditions.
The model is automatically built using IBM AutoAI, which generates multiple pipelines, selects the best model, and deploys it as an API service.
The final deployed model predicts crop yield values that can help farmers and agricultural planners make better decisions.

2️⃣ Project Features

✅ Automatic machine learning model generation

✅ Data preprocessing and feature engineering

✅ Hyperparameter optimization

✅ Multiple pipeline generation

✅ Best model selection using evaluation metrics

✅ Online model deployment

✅ Real-time prediction using API

**🔧 Technologies **

Python – Used for data processing and building the machine learning model.

IBM Cloud Pak for Data – Platform used to develop, train, and deploy the ML model.

IBM AutoAI – Automatically generates ML pipelines, selects the best model, and optimizes performance.

Jupyter Notebook – Used for running experiments and handling the dataset.

IBM Watson Machine Learning – Used to deploy the trained model as an API service.

GitHub – Used to store and share the project code and files.

Snap Boosting Machine Regressor – Algorithm used to predict crop yield values.
